Om denna applikation

Denna programöversikt (version 2024) är skapad för ett modernt, luftigt gränssnitt med fokus på användarvänlighet och estetik. Applikationen använder Tailwind CSS, mörkt/ljust läge, och ett fräscht typsnitt för en upplevelse som matchar vår tids designstandard.

  • Modern design med Tailwind CSS
  • Mörk/Ljus läge
  • Sökfunktion för snabb filtrering
  • Programdata från JSON för enkel uppdatering

Allt för att upplevas som en modern 2024-applikation, långt ifrån 70-talet.
